✨ Crown Lamp

An interactive, jewel-studded crown with multicolor LEDs

Project Description

Made during the IMA Low-Res Summer session, Crown Lamp is a playful, wearable light sculpture. Each spike has an LED, and a distance sensor changes the colors (white/green/blue) depending on how close it is while worn. I air-brushed the 3D-printed crown with blue-green gradients and added rhinestones for extra sparkle.

Tools: Arduino + sensors, 3D printing, spray paint, rhinestones.
